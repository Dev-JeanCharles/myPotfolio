Sobre mim

Me chamo Jean Charles Duarte Santos, tenho 18 anos, moro no Rio de Janeiro, sou desenvolvedor full-stack e comecei nessa carreira como programador há 3 anos atrás, além disso eu tenho conhecimento em: Java, Python, HTML e CSS, JavaScript, Spring Framework, React Framework e AWS Cloud.

Comecei nessa jornada atuando como programador por meio do curso técnico de informática o qual realizei e tive minhas primeiras experiências com a linguagem Python, uma linguagem o qual apaixonei logo de cara, aprendi a lógica de programação e comecei a fazer uso de frameworks para implementar minhas primeiras API'S REST com o framework Django. Após ter terminado o curso iniciei meus estudos para aprender novas tecnologias e linguagens como o Java e Spring Framework onde comecei a entender o universo por trás dessas tecnologias que são tão almejadas no mercado de trabalho.

Atualmente estou aprendendo novos assuntos e me aprofundando nos que já possuo conhecimento.
